Related Doc: package types

class LazyValue extends Value

An abstract node in a FaunaDB response tree. An instance of this class does not have any accessible data. It must first be coerced into a concrete type.

Annotations
@JsonDeserialize()
Example:

  1. Consider the ResponseNode modeling the root of the following tree:

    {
  "ref": {"@ref": "some/ref" },
  "data": { "someKey": "string1", "someKey2": 123 }
}

    The data in this tree can be accessed using:

    node.asObject("ref").asRef // Ref("some/ref")
node.asObject("data").asObject("someKey").asString // "string1"
